    ดึงรายการสินค้า¶

    สิ่งที่จะได้เรียนรู้¶

    • สร้าง API ดึงสินค้าทั้งหมด
    • Pagination (แบ่งหน้า)
    • Filter ตาม Category

    Pagination คืออะไร?¶

    Pagination คือการแบ่งข้อมูลเป็นหน้าๆ ไม่ดึงทั้งหมดในครั้งเดียว

    ทำไมต้อง Pagination?¶

    แบบ ข้อมูล 10,000 รายการ
    ไม่มี Pagination โหลดช้า, เปลือง memory
    มี Pagination โหลดทีละ 10-20 รายการ, เร็ว

    ตัวอย่าง URL¶

    GET /api/products?page=1&limit=10    → สินค้ารายการ 1-10
GET /api/products?page=2&limit=10    → สินค้ารายการ 11-20
GET /api/products?page=3&limit=10    → สินค้ารายการ 21-30

    SQL LIMIT และ OFFSET¶

    -- ดึง 10 รายการแรก
SELECT * FROM products LIMIT 10;

-- ดึง 10 รายการ เริ่มจากรายการที่ 11 (ข้าม 10 รายการแรก)
SELECT * FROM products LIMIT 10 OFFSET 10;

-- สูตร: OFFSET = (page - 1) * limit
-- Page 1: OFFSET = (1-1) * 10 = 0
-- Page 2: OFFSET = (2-1) * 10 = 10
-- Page 3: OFFSET = (3-1) * 10 = 20

    สร้างไฟล์ get-all.php¶

    โครงสร้างไฟล์¶

    backend-mysql/
└── api/
    └── products/
        └── get-all.php    ← สร้างไฟล์นี้

    โค้ดสมบูรณ์¶

    <?php
/**
 * Get All Products API
 * GET /api/products/get-all.php
 *
 * Query params:
 * - page: หน้าที่ต้องการ (default: 1)
 * - limit: จำนวนต่อหน้า (default: 10)
 * - category_id: กรองตามหมวดหมู่
 * - search: ค้นหาชื่อสินค้า
 */

require_once '../../config/database.php';

// ต้องเป็น GET
if ($_SERVER['REQUEST_METHOD'] !== 'GET') {
    sendResponse(405, 'Method not allowed. Use GET.');
}

// รับ query parameters
$page = max(1, (int)($_GET['page'] ?? 1));
$limit = min(50, max(1, (int)($_GET['limit'] ?? 10)));
$categoryId = $_GET['category_id'] ?? null;
$search = $_GET['search'] ?? null;

// คำนวณ offset
$offset = ($page - 1) * $limit;

try {
    // ===================================
    // สร้าง SQL Query
    // ===================================

    $sql = "
        SELECT
            p.id,
            p.name,
            p.description,
            p.price,
            p.stock,
            p.image_url,
            p.is_active,
            p.created_at,
            c.id AS category_id,
            c.name AS category_name
        FROM products p
        LEFT JOIN categories c ON p.category_id = c.id
        WHERE p.is_active = 1
    ";

    $params = [];

    // Filter by category
    if ($categoryId) {
        $sql .= " AND p.category_id = ?";
        $params[] = $categoryId;
    }

    // Search by name
    if ($search) {
        $sql .= " AND p.name LIKE ?";
        $params[] = "%$search%";
    }

    // ===================================
    // นับจำนวนทั้งหมด (สำหรับ pagination info)
    // ===================================

    $countSql = "SELECT COUNT(*) as total FROM products p WHERE p.is_active = 1";
    if ($categoryId) {
        $countSql .= " AND p.category_id = ?";
    }
    if ($search) {
        $countSql .= " AND p.name LIKE ?";
    }

    $countStmt = $pdo->prepare($countSql);
    $countStmt->execute($params);
    $totalItems = (int) $countStmt->fetch()['total'];
    $totalPages = ceil($totalItems / $limit);

    // ===================================
    // ดึงข้อมูลพร้อม Pagination
    // ===================================

    $sql .= " ORDER BY p.id DESC LIMIT ? OFFSET ?";
    $params[] = $limit;
    $params[] = $offset;

    $stmt = $pdo->prepare($sql);
    $stmt->execute($params);
    $products = $stmt->fetchAll();

    // ===================================
    // ส่ง Response
    // ===================================

    sendResponse(200, 'Products retrieved successfully', [
        'products' => $products,
        'pagination' => [
            'current_page' => $page,
            'per_page' => $limit,
            'total_items' => $totalItems,
            'total_pages' => $totalPages,
            'has_next' => $page < $totalPages,
            'has_prev' => $page > 1
        ]
    ]);

} catch (PDOException $e) {
    sendResponse(500, 'Database error: ' . $e->getMessage());
}
?>

    อธิบายโค้ด¶

    1. รับ Query Parameters¶

    $page = max(1, (int)($_GET['page'] ?? 1));
$limit = min(50, max(1, (int)($_GET['limit'] ?? 10)));
    • max(1, ...) - ไม่ให้ต่ำกว่า 1
    • min(50, ...) - ไม่ให้เกิน 50 (ป้องกันโหลดข้อมูลเยอะ)

    2. Dynamic WHERE Clause¶

    $sql = "SELECT ... WHERE p.is_active = 1";
$params = [];

if ($categoryId) {
    $sql .= " AND p.category_id = ?";
    $params[] = $categoryId;
}

    เพิ่มเงื่อนไขตามที่ส่งมา

    3. นับจำนวนทั้งหมด¶

    $countSql = "SELECT COUNT(*) as total FROM products p WHERE ...";
$totalItems = (int) $countStmt->fetch()['total'];
$totalPages = ceil($totalItems / $limit);

    ใช้คำนวณจำนวนหน้าทั้งหมด

    Query Parameters สรุป¶

    Parameter Type Default คำอธิบาย
    page integer 1 หน้าที่ต้องการ
    limit integer 10 จำนวนต่อหน้า (max 50)
    category_id integer - กรองตามหมวดหมู่
    search string - ค้นหาชื่อสินค้า
