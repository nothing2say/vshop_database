    PostgreSQL + Prisma¶

    ภาพรวม¶

    PostgreSQL เป็น SQL Database ขั้นสูง และ Prisma เป็น Modern ORM ที่ให้ Type-safety

    MySQL vs PostgreSQL¶

    MySQL PostgreSQL
    License Dual License Open Source
    ACID รองรับ รองรับดีกว่า
    JSON รองรับ รองรับดีกว่า
    Extensions จำกัด เยอะมาก
    Performance Read-heavy Complex queries

    Prisma คืออะไร?¶

    Prisma เป็น ORM (Object-Relational Mapping) รุ่นใหม่ที่ให้ Type-safety และ Auto-completion

    ข้อดีของ Prisma¶

    Feature คำอธิบาย
    Type-safe TypeScript types อัตโนมัติ
    Auto-complete VS Code แนะนำโค้ด
    Migration จัดการ schema changes
    Prisma Studio GUI ดูข้อมูล

    เทคโนโลยีที่ใช้¶

    Tool หน้าที่
    PostgreSQL Database
    Node.js Runtime
    TypeScript Type-safe JavaScript
    Express.js Web Framework
    Prisma ORM

    ตัวอย่างโค้ด¶

    Prisma Schema¶

    // prisma/schema.prisma

datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}

generator client {
  provider = "prisma-client-js"
}

model User {
  id        Int      @id @default(autoincrement())
  email     String   @unique
  name      String
  role      Role     @default(CUSTOMER)
  orders    Order[]
  cart      Cart[]
  createdAt DateTime @default(now())
}

model Product {
  id          Int       @id @default(autoincrement())
  name        String
  price       Decimal
  stock       Int       @default(0)
  category    Category  @relation(fields: [categoryId], references: [id])
  categoryId  Int
  isActive    Boolean   @default(true)
}

model Category {
  id       Int       @id @default(autoincrement())
  name     String
  products Product[]
}

enum Role {
  CUSTOMER
  ADMIN
}

    API Endpoint¶

    // routes/products.ts
import { Router } from 'express';
import { PrismaClient } from '@prisma/client';

const router = Router();
const prisma = new PrismaClient();

// GET all products
router.get('/', async (req, res) => {
    try {
        const products = await prisma.product.findMany({
            where: { isActive: true },
            include: { category: true },
            orderBy: { id: 'desc' }
        });

        res.json({
            status: 200,
            message: 'Products retrieved',
            data: products
        });
    } catch (error) {
        res.status(500).json({
            status: 500,
            message: error.message
        });
    }
});

// GET product by ID - Type-safe!
router.get('/:id', async (req, res) => {
    const { id } = req.params;

    const product = await prisma.product.findUnique({
        where: { id: parseInt(id) },
        include: { category: true }
    });

    if (!product) {
        return res.status(404).json({
            status: 404,
            message: 'Product not found'
        });
    }

    res.json({
        status: 200,
        data: product
    });
});

export default router;

    Type-safe Query¶

    // Auto-complete ทำงาน!
const user = await prisma.user.findUnique({
    where: { email: 'test@email.com' },
    select: {
        id: true,
        name: true,
        orders: {
            include: {
                items: {
                    include: {
                        product: true
                    }
                }
            }
        }
    }
});

// user.name -> string
// user.orders -> Order[]
// user.orders[0].items[0].product.name -> string

    Migration¶

    # สร้าง migration จาก schema
npx prisma migrate dev --name init

# Apply migration ไป production
npx prisma migrate deploy

# เปิด Prisma Studio (GUI)
npx prisma studio

    เมื่อไหร่ควรใช้ Prisma?¶

    เหมาะกับ:

    • โปรเจค TypeScript
    • ต้องการ Type-safety
    • ต้องการ Auto-complete
    • Team ใหญ่

    ไม่เหมาะกับ:

    • โปรเจคเล็กๆ
    • ไม่คุ้นเคย TypeScript
    • ต้องการ raw SQL ซับซ้อน

    • TypeScript Handbook

    🚀 วิธีการรัน¶

    ขั้นตอนที่ 1: ติดตั้ง PostgreSQL¶

    brew install postgresql@15
brew services start postgresql@15
createdb vshop

    ดาวน์โหลดจาก postgresql.org และติดตั้ง

    docker run --name vshop-postgres \
  -e POSTGRES_PASSWORD=password \
  -e POSTGRES_DB=vshop \
  -p 5432:5432 \
  -d postgres:15
    1. สมัครที่ supabase.com
    2. สร้าง Project ใหม่
    3. ไปที่ Settings → Database → Connection String

    ขั้นตอนที่ 2: รัน Backend¶

    cd VShop_Workshop/prisma-nextjs/starter/backend
npm install
cp .env.example .env
# แก้ไข DATABASE_URL ใน .env

npx prisma generate
npx prisma migrate dev --name init
npm run dev

    Backend พร้อมใช้งาน

    • API: http://localhost:3002
    • Prisma Studio: npx prisma studio (http://localhost:5555)

    ขั้นตอนที่ 3: รัน Frontend¶

    cd VShop_Workshop/prisma-nextjs/starter/frontend
npm install
npm run dev

    Frontend พร้อมใช้งาน

    Web: http://localhost:3000

    🗄️ สร้างข้อมูลตัวอย่าง¶

    รัน seed script:

    cd VShop_Workshop/prisma-nextjs/starter/backend
npm run prisma:seed

    หรือเพิ่มข้อมูลผ่าน Prisma Studio:

    npx prisma studio
